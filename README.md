# GymPoseDetection

## Getting Started

### **Server Setup (FastAPI)**

1. **Create a Virtual Environment**:

   ```bash
   python3 -m venv venv
   ```

2. **Activate the Virtual Environment**:

   - **macOS/Linux**:
     ```bash
     source venv/bin/activate
     ```
   - **Windows**:
     ```bash
     .\venv\Scripts\activate
     ```

3. **Install Required Dependencies**:

   ```bash
   pip install -r requirement.txt
   ```
